# persian-movie-db
پروژه پایگاه داده فیلم‌های ایرانی 
این پروژه با هدف طراحی و پیاده‌سازی یک پایگاه داده رابطه‌ای نرمال‌شده بر اساس دیتاست فیلم‌های ایرانی انجام شده است.
پایگاه داده شامل سه جدول Movies، Genres و MovieGenres می‌باشد که رابطه چندبه‌چند بین فیلم‌ها و ژانرها را پیاده‌سازی می‌کند. ساختار جداول تا سطح فرم نرمال سوم (3NF) نرمال‌سازی شده است.
